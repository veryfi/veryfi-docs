## Add a Tag
Add a tag to a document.

### HTTP Request
`PUT ENVIRONMENT_URL/api/v8/partner/documents/DOCUMENT_ID/tags/`

### Parameters
| Field | Type   | Required | Description           |
|-------|--------|----------|-----------------------|
| name  | String | Y        | 			Name of tag to add |
